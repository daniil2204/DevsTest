Backend:
Nest, Typescript, Axios.

Frontend:
Next, Typescript, SASS, React Query, Axios.

Project commands
cd .\backend\
yarn - install dependencies
yarn start:dev - run backend
port - 4000 | 8000
cd .\frontend\
yarn - install dependencies
yarn dev - run frontend
port - 3000
